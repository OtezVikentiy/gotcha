## [0.1.0] - 2026-07-22

Начальный набор возможностей self-hosted релиза:

### Added

- **Проблемы / трекинг ошибок**: приём событий, совместимый с Sentry SDK,
  автоматическая группировка в проблемы (issues), стек-трейсы, хлебные крошки
  (breadcrumbs), теги/контексты.
- **Производительность / трейсинг**: распределённые трейсы и транзакции, Web
  Vitals, обнаружение проблем производительности и обнаружение регрессий.
- **Метрики**: приём метрик OTLP, запросы по метрикам, правила оповещений по
  порогам и инциденты.
- **Профилирование**: профили CPU/флеймграфы из payload'ов профилирования Sentry
  и pprof, с обнаружением регрессий профилей.
- **Мониторинг аптайма**: HTTP-проверки из встроенного локального региона или с
  удалённых проб (`--mode=probe`), обнаружение инцидентов, публичные страницы
  статуса.
- **Оповещения**: доставка по email, вебхуку и в Telegram; правила для новых
  проблем, всплесков, порогов по метрикам и регрессий производительности/аптайма.
- **Организации, команды и RBAC**: мультитенантные организации, проекты и роли
  участников.
- **SSO**: вход через generic OIDC, Yandex ID и VK ID, каждый настраивается
  независимо.
- **Значения по умолчанию для приватности и безопасности**: серверная очистка
  PII (обнуление IP/email, редактирование по ключам) и защита от SSRF для
  исходящих запросов вебхуков/аптайма, обе включены по умолчанию.
- **Настраиваемый срок хранения** по каждому типу сигнала (события/транзакции,
  спаны, метрики, профили) через TTL ClickHouse.
- **Настройки производительности на уровне проекта** и переопределения срока
  хранения спанов.
- Развёртывание одним бинарником (`gotcha`) с
  `--mode=ingest|web|uptime|probe|all`.
- Документация внутри продукта (`/docs`) на английском и русском.
- Файлы open-source проекта: README, LICENSE (Apache-2.0), CONTRIBUTING,
  SECURITY, CODE_OF_CONDUCT, `.env.example`.
- Версия сборки в подвале интерфейса, на странице «О программе», в
  `--version` и `/version`.
