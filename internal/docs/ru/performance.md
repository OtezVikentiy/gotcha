# Производительность

Раздел «Производительность» показывает эндпойнты приложения с перцентилями латентности (p50/p75/p95/p99), трафиком и индексом Apdex.

## Эндпойнты и Apdex
Список эндпойнтов сортируется по трафику или латентности; Apdex считает долю запросов, уложившихся в порог удовлетворительного времени ответа T. Резкий рост p95/p99 при стабильном трафике обычно указывает на деградацию, а не на нагрузку.

## Деталь эндпойнта
На странице эндпойнта — гистограмма латентности, график по времени и список самых медленных трейсов с переходом к связанным проблемам, если запрос завершился ошибкой.

## Web Vitals
Отдельная вкладка показывает метрики скорости страниц в браузере пользователя — LCP, INP, CLS, FCP, TTFB — с перцентилем p75 и рейтингом good/needs-improvement/poor по каждой странице.

## Регрессии
Проблемы производительности — это автоматически обнаруженные регрессии: статистически значимое ухудшение латентности или Web Vitals относительно базовой линии за предыдущий период.
