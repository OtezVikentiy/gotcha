# Первые шаги

Gotcha — платформа наблюдаемости: ошибки, производительность, метрики, аптайм и оповещения в одном месте.

## 1. Создайте проект
Проект — это отдельное приложение или сервис. Ошибки, транзакции и метрики привязаны к проекту. Организация объединяет проекты и участников.

## 2. Подключите SDK
На странице «Настройки проекта → Подключение» есть DSN проекта и сниппеты. Gotcha принимает данные по протоколу приёма Sentry — используйте официальный Sentry SDK вашего языка, указав DSN Gotcha. Подробнее: [SDK и интеграции](/docs/sdk).

## 3. Получите первое событие
После подключения вызовите ошибку в приложении — она появится в разделе «Проблемы». Одинаковые ошибки группируются в одну «проблему».

## 4. Настройте оповещения
В «Оповещениях» проекта добавьте канал доставки (email/webhook/Telegram) и включите правила — и вы узнаете о новых проблемах и всплесках.
