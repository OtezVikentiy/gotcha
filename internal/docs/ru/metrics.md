# Метрики

Раздел «Метрики» хранит числовые временные ряды, которые ваше приложение отправляет через SDK или по протоколу OTLP.

## Типы и агрегации
Метрика может быть counter (счётчик, только растёт), gauge (произвольное значение в моменте) или histogram (распределение значений). Для просмотра доступны агрегации p50/p95/avg/sum за выбранный период.

## Фильтры и лейблы
Каждая метрика может нести лейблы (например, окружение, регион, версия релиза). Список метрик фильтруется по окружению, периоду и значениям лейблов, что позволяет сравнивать один и тот же показатель в разных срезах.

## Оповещения по метрикам
На метрику можно повесить правило оповещения — порог по значению агрегации за интервал; при превышении срабатывает канал доставки, настроенный в «Оповещениях».
